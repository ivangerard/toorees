# Toorees

Toorees it's an application that help tourist to find their destination Easy and Safe !

No worries to become a tourist.

## Architecture For Backend
1. Express
2. Mongoose , MongoDB
3. Mocha, Chai

## Architecture For Frontend

1. HTML
2. CSS
3. AJAX & jQuery
4. Gulp
5. Webpack
